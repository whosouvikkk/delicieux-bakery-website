🥐 Délicieux Bakery Website

A modern, fully responsive bakery website with premium UI/UX, smooth animations, and a functional backend for orders and table reservations.

--------------------------------------------------

🚀 Features

🎨 Frontend
- Fully responsive design (mobile + tablet + desktop)
- Smooth scroll animations & reveal effects
- Interactive UI with hover effects & micro-interactions
- Real bakery food images (no illustrations)
- Dynamic menu section with pricing
- Order Now & Book Table modals
- Cart interaction with toast notifications

⚙️ Backend
- REST API built with Express.js
- Order management system
- Reservation (table booking) system
- Menu API with filtering
- Order status tracking (auto updates)
- Availability checking for booking slots

📦 API Features
- GET /api/menu → fetch menu items
- POST /api/orders → place an order
- GET /api/orders/:id → track order
- POST /api/reservations → book a table
- GET /api/availability → check slots

--------------------------------------------------

⚡ Installation & Setup

1. Clone the repo:
git clone

2. Navigate:
cd delicieux-bakery-website

3. Install dependencies:
npm install

4. Run backend:
npm start

Server runs at:
http://localhost:3001
